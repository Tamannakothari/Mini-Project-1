# Mini-Project-1
This is the repository for my mini project 1, where I will be creating a website for a client

# AMAN Babson Website

# Client
AMAN – South Asian Student Association at Babson College

# Description
This website was created for AMAN, Babson’s South Asian Student Association. The goal of the site is to provide a central platform where students can learn about AMAN, explore events and performances, view the executive board, and find ways to get involved.

The site includes:
- A homepage with an introduction and embedded AMAN show video  
- An About page explaining AMAN’s mission and activities  
- A Leadership page featuring the Spring 2026 executive board  
- A Countries page highlighting South Asian countries that AMAN represents 
- A Gallery page showcasing Fall 2025 show moments  
- A Join AMAN page with a form for new members  

# Live Site
https://tamannakothari.github.io/Mini-Project-1/ 

# What I Learned
Through this project, I learned how to build a website for a real client rather than for myself. This required thinking about user needs, navigation, and clarity instead of just design.

I also learned:
- How to structure a multi-page website using semantic HTML  
- How to create consistent styling across pages using an external CSS file  
- How to embed external media like YouTube videos   
- How to translate feedback and ideas from a group (the AMAN e-board) into an actual website  

This project helped me understand the importance of planning, user experience, and building something functional and meaningful for a real audience.
